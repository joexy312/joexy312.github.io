On the Automatic Generation of Medical Imaging Reports

Abstract

医学影像在临床诊断和治疗中被广泛应用，但手写报告面临容易出错、耗时等问题;
本文研究医学影像报告的自动生成。所面临的主要挑战：

  • 一份完整报告包含多种形式的信息;
  • 医学图像中的异常区域很难识别;
  • 报告通常非常长
    本文贡献：
  • 构建了一个多任务学习框架，联合执行标签预测和段落生成
  • 提出一种共同注意机制来定位包含异常的区域，并为它们生成叙述
  • 开发一个分层的LSTM模型来生成长段落。
    作者在两个公开的数据集上证明了所提出方法的有效性

Introduction

在阅读医学影像时，专家撰写文字报告来叙述成像研究中检查的身体每个区域的发现，特别是每个区域是否被发现正常、异常或潜在异常。

图中，findings列举影像学研究中检查的身体各个区域的放射学观察结果，tags部分列出了代表调查结果中关键信息的关键字。

动机：对于经验不足的放射科医生和病理科医生，尤其是那些在医疗质量相对较低的农村地区工作的医生来说，撰写医学影像报告是一项艰巨的任务。而对于经验丰富的放射科医生和病理科医生来说，撰写影像报告既繁琐又耗时。因此，本文希望研究是否能自动生成医学图像的报告。

挑战：

  • 完整的诊断报告包含多种形式的信息，在统一框架中生成这种异构信息在技术上要求很高。解决方法：建立多任务(Multi-task)框架，将tags的预测视为多标签分类任务，将长的描述语句的生成视为文本生成任务。
  • 如何将图像区域和正确的描述语句对应起来也是有挑战性的。解决方法:引入共同注意力机制来解决这些问题，该机制同时关注图像和预测标签，并探索视觉和语义信息的协同效应。
  • 报告中的语句很长，包含多个句子。解决方法：通过多级LSTM来生成长文本，与注意力机制联合工作，首先生成topics，然后根据不同的topics生成细粒度的描述。

Related Works

  • Textual Labeling of medical images
    在将文本和医学图像联系起来的相关研究中，目标文本大多都是结构化或者半结构化的文本(e.g. tags,templates)，而不是自然文本。相关研究通常构建端到端的管道进行预测，比如CNN-RNN等网络结构。
    但在现实世界中，收集、处理半结构化、结构化数据比处理自然数据的实用性更低。本文似乎是首次研究生成真实的非结构化的报告。
  • Image captioning with deep learning
    Image captioning的目标是自动根据图像生成文本描述。相关研究很多都是基于CNN-RNN网络。最近，基于注意力机制的方法在image captioning任务中展现除了良好的效果。

On the Automatic Generation of Medical Imaging Reports

DeepOpht: Medical Report Generation for Retinal Images via Deep Models and Visual Explanation基于深度模型和视觉解释的视网膜图像医学报告生成

Abstract

本文提出了一种基于人工智能的方法，旨在改进传统的视网膜疾病治疗程序，并帮助眼科医生提高诊断效率和准确性。

该网络包含一个DNN-based module和一个DNN visual explanation module。其中，DNN-based module包含一个视网膜疾病判别器和临床诊断生成器。
本文同时还提出了一个大范围的视网膜疾病图像数据集，并进行了手工标注。本文提出的方法可以有效的生成有价值的视网膜图像诊断信息和视觉解释。

Introduction

人工智能的当前技术水平涉及深度学习研究，本文认为，深度学习是帮助眼科医生和改进传统视网膜疾病治疗程序的有前途的方法之一。
本文提出了一种基于人工智能的方法，通过输入视网膜图像自动生成医学报告，旨在优化传统的视网膜疾病诊治流程，提升诊断的有效性和准确性。主要的方法就是利用基于深度学习的模型，包括视网膜疾病判别器retinal disease identifier (RDI)和临床诊断生成器clinical description generator (CDG)，让传统诊疗过程部分自动化，以此提高效率。
本文还提出了DeepEyeNet数据集用于网络的训练和验证。

在传统方法(a)中，大部分工作都需要由医生人工完成;本文提出的基于人工智能的方法(b)中，主要包括DNN-based和DNN visual explanation模块，DNN-based模块的输出是疾病分类“A”和临床文本描述。DNN视觉解释模块将来自DNN模块的分类任务信息可视化。

Methodlogy

DNN-based Module包含RDI和CDG两个部分：
Retinal Disease Identifier（RDI）：本文提出了分别基于VGG和ResNet的深度学习模型，在ImageNet上进行预训练，然后在DEN数据集上进行训练。通过对预训练模型的迁移可以有效的帮助提取低级的特征信息。因此，本文利用预训练模型来优化模型的效果。
Clinical Description Generator (CDG)：为了生成针对输入图像的临床文本描述，本文使用了基于CNN的预训练模型包括MobileNetV2、VGG16、VGG19、InceptionV3等作为图像的编码器，然后利用LSTM作为解码器来生成文本。当试图通过LSTM单元生成临床描述时，本文还结合了beam-search来获得更好的最终输出描述。在眼科实践中，普遍存在的关键词具有无序的性质，可以帮助眼科医生创建医疗报告。受此启发，本文利用关键词来强化CDG子模块。因此，网络具有文本和图像两个输入，本文用均值法来融合两种特征。

DNN Visual Explanation Modules：
为了改进传统的视网膜疾病治疗程序，本文在提出的基于人工智能的方法中加入了DNN视觉解释模块。此外，本文利用这个模块来帮助验证方法的有效性，具体可以参考论文的实验部分。

Automated radiology report generation using conditioned transformers

Abstract

医院的放射学报告撰写是一项耗时的任务，也需要相关的放射学家的经验。本文提出了一种深度学习模型，在给定来自公共IU-Xray数据集的chestx射线图像的情况下，自动生成放射学报告。
本文的工作包含三个阶段：

  • 对预训练模型Chexnet进行微调，预测对应图像的tags。
  • 根据预测tags的预训练嵌入计算加权语义特征。
  • 在视觉和语义特征上调节预先训练的GPT2模型，以生成完整的医学报告。
    对生成报告的评估：基于词重叠的指标word-overlap metrics，同时加入了i基于语义的相似度指标。
