Pytorch踩坑记录

记录一下Pytorch报过的比较常见的错误

IndexError: Dimension out of range (expected to be in range of [-1, 0], but got 1)

一般还是因为预测值的维度和真实值的维度不匹配，可以打印一下tensor.size()看看
可以参考github上的一个issue

RuntimeError: Expected object of scalar type Double but got scalar type Float for argument #2 ‘mat2’

按照提示是数据类型的原因，查看输入数据类型为torch.float64，将输入数据类型转为torch.float32，搞定。
